SolydXK System Settings: Device Driver Help

Descripción

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Importante

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Comandos

Si prefiere instalar los drivers a través de la terminal, puede utilizar los siguientes comandos:

 

# ddm -i amd

This will install the drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Esto seleccionará los drivers apropiados para su tarjeta gráfica Nvidia. En caso de tener una tarjeta híbrida (Nvidia e Intel al mismo tiempo) se seleccionará el driver Bumblebee. Utilice "ddm -p nvidia" para quitar los drivers de su sistema.

# ddm -i open

Puede utilizar este comando para volver a los drivers Nouveau. Ésto quitará cualquier driver propietario de su sistema.

# ddm -i broadcom

Éste comando instalará los drivers apropiados para su adaptador wireless Broadcom. Utilice "ddm -p broadcom" para quitar los drivers de su sistema.

# ddm -i pae

Ésto instalará el kernel PAE en sistemas multi-procesador que ejecuten un SO de 32-bits. Utilice "ddm -p pae" para quitar el kernel PAE de su sistema. Nótese que no podrá remover el kernel si éste fue iniciado con el sistema.
