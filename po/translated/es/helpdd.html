SolydXK System Settings: Ayuda para el controlador del dispositivo

Descripción

Device Driver es una herramienta que ayuda a los usuarios a instalar controladores de hardware de terceros. Actualmente se pueden instalar los controladores de Nvidia, AMD/ATI y Broadcom y permite instalar el kernel PAE para sistemas multinúcleo de 32 bits. Device Driver sólo listará aquellos dispositivos que son soportados. No listará ningún otro dispositivo que no haya sido mencionado anteriormente.

Importante

Device Driver sólo puede instalar los controladores que están disponibles en los repositorios. Asegúrese de tener una conexión a Internet si desea instalar los controladores.

Comandos

Si prefiere instalar los controladores mediante una terminal, puede utilizar los siguientes comandos:

 

# ddm -i amd

Esto instalará los controladores para su tarjeta gráfica AMD/ATI. Las tarjetas AMD/ATI a partir de la versión HD 5000 son compatibles con los controladores. Las versiones entre 1000 y 5000 utilizarán los controladores Radeon. Utilice "ddm -p amd" para eliminar los controladores de su sistema.

# ddm -i nvidia

Esto seleccionará los controladores adecuados para su tarjeta gráfica Nvidia. Seleccionará los controladores Bumblebee en caso de que tengas una tarjeta híbrida (tanto Nvidia como Intel). Utilice "ddm -p nvidia" para eliminar los controladores de su sistema.

# ddm -i open

Puede utilizar este comando si desea volver a los controladores abiertos de Nouveau. Eliminará cualquier controlador propietario de su sistema.

# ddm -i broadcom

Este comando instalará el controlador adecuado para su adaptador inalámbrico Broadcom. Utilice "ddm -p broadcom" para eliminar los controladores de su sistema.

# ddm -i pae

Esto instalará el kernel PAE en sistemas multiprocesadores que ejecuten un SO de 32 bits. Utilice "ddm -p pae" para eliminar el kernel PAE de su sistema. Tenga en cuenta que no puede eliminar el kernel cuando está arrancando en ese kernel.
