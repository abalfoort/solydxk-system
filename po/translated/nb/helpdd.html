SolydXK System Settings: Device Driver Help

Beskrivelse

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Viktig

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Kommandoer

Hvis du foretrekker å installere driverne via en terminal, kan du bruke følgende kommandoer:

 

# ddm -i amd

This will install the drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Dette vil velge de riktige driverne for ditt Nvidia-kort. Det vil velge Bumblebee-drivere dersom du har et hybrid-kort (både Nvidia og Intel). Bruk "ddm -p nvidia" for å fjerne driverne fra systemet.

# ddm -i open

Du kan bruke denne kommandoen hvis du ønsker å tilbakestille systemet til de åpne Nouveau-driverne. Det vil fjerne alle proprietære driver fra systemet ditt.

# ddm -i broadcom

Denne kommandoen vil installere den riktige driveren for ditt trådløse Broadcom-adapter. Bruk "ddm -p broadcom" for å fjerne driverne fra systemet ditt.

# ddm -i pae

Dette vil installere PAE-kjernen på multi-prosessor systemer som kjører et 32-bits OS. Bruk "ddm -p pae" for å fjerne PAE-kjernen fra systemet. Merk at du ikke kan fjerne kjernen når du har startet opp med denne.
