Definições do sistema SolydXK: Ajuda do controlador de dispositivo

Descrição

O Device Driver é uma ferramenta que ajuda os utilizadores a instalar controladores de ‘hardware’ de terceiros. Atualmente, podem ser instalados controladores Nvidia, AMD/ATI e Broadcom e permite instalar o kernel PAE para sistemas multi-core de 32 bits. O Device Driver apenas irá listar os dispositivos suportados. Ele não listará nenhum outro dispositivo além dos mencionados acima.

Importante

O Controlador de Dispositivos só pode instalar controladores que estejam disponíveis nos repositórios. Certifique-se de que tem uma ligação à Internet se quiser instalar os controladores.

Comandos

Se preferir instalar os controladores através do terminal, pode utilizar os seguintes comandos:

 

# ddm -i amd

Isto irá instalar os controladores para a sua placa gráfica AMD/ATI. As placas AMD/ATI a partir da versão HD 5000 são suportadas pelos controladores. As versões entre 1000 e 5000 utilizarão os controladores Radeon. Utilize “ddm -p amd” para remover os controladores do seu sistema.

# ddm -i nvidia

Isto irá selecionar os controladores corretos para a sua placa gráfica Nvidia. Selecionará os controladores Bumblebee no caso de ter uma placa híbrida (tanto Nvidia como Intel). Utilize “ddm -p nvidia” para remover os controladores do seu sistema.

# ddm -i open

Pode utilizar este comando se pretender reverter para os controladores Nouveau abertos. Irá remover quaisquer controladores proprietários do seu sistema.

# ddm -i broadcom

Este comando irá instalar o controlador correto para o seu adaptador Broadcom sem fios. Use “ddm -p broadcom” para remover os controladores do seu sistema.

# ddm -i pae

Isto irá instalar o kernel PAE em sistemas multi-processador a correr um SO de 32-bit. Utilize “ddm -p pae” para remover o kernel PAE do seu sistema. Note que não pode remover o kernel quando arranca atualmente nesse kernel.
