SolydXK System Settings: Device Driver Help

Açıklama

The Device Driver is a tool to help users installing third-party hardware drivers. Currently Nvidia, AMD/ATI and Broadcom drivers can be installed and allows to install the PAE kernel for multi-core 32-bit systems. The Device Driver will only list those devices that are supported. It will not list any other devices then mentioned above.

Önemli

The Device Driver can only install drivers which are available in the repositories. Make sure you have an internet connection if you want to install the drivers.

Komutlar

Eğer sürücüleri uçbirim yardımıyla kurmak isterseniz, aşağıdaki komutları kullanabilirsiniz:

 

# ddm -i amd

This will install the drivers for your AMD/ATI graphical card. AMD/ATI cards from version HD 5000 and up are supported by the drivers. Versions between 1000 and 5000 will use the Radeon drivers. Use "ddm -p amd" to remove the drivers from your system.

# ddm -i nvidia

Bu Nvidia grafik kartı için doğru sürücüyü seçecektir. (Nvidia + Intel) hibrit kartlar olması durumunda Bumblebee seçilecektir. Sürücüyü sistemden silmek için "ddm -p nvidia" komutunu kullanınız.

# ddm -i open

Açık kaynak Nouveau sürücüsüne dönmek isterseniz bu komutu kullanabilirsiniz. Bu tüm sahipli sürücüyü sisteminizden siler.

# ddm -i broadcom

Bu komut Broadcom kablosuz ağ sürücüsü için doğru sürücüyü seçecektir. Sürücüyü sistemden silmek için "ddm -p broadcom" komutunu kullanınız.

# ddm -i pae

Bu çoklu-çekirdek üstünde çalışan 32 bit işletim sistemi için PAE çekirdeği kurar. PAE çekirdeği sistemden silmek için "ddm -p pae" komutunu kullanınız. Eğer şu an sisteminizi PAE çekirdek ile açtı iseniz silemezsiniz.
