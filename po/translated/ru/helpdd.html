SolydXK System Settings: Устройство Драйвер Помощь

Описание

Драйвер устройства — это инструмент, помогающий пользователям устанавливать драйвера оборудования сторонних производителей. В настоящее время доступны драйвера Nvidia, AMD/ATI и Broadcom, которые позволяют установить ядро ​​PAE для многоядерных 32-битных систем. В драйвере устройства будут перечислены только те устройства, которые поддерживаются. В нем не будут перечислены другие устройства, упомянутые выше.

Важно

Драйвер устройства может устанавливать только те драйвера, которые доступны в репозиториях. Убедитесь, что у вас есть подключение к Интернету, если вы хотите установить драйвера.

Команды

Если вы предпочитаете устанавливать драйвера через терминал, вы можете использовать следующие команды:

 

# ddm -i amd

При этом будут установлены драйвера для вашей графической карты AMD/ATI. Карты AMD/ATI версии HD 5000 и выше поддерживаются драйверами. Версии между 1000 и 5000 будут использовать драйвера Radeon. Используйте «ddm -p amd», чтобы удалить драйвера из вашей системы.

# ddm -i nvidia

Это позволит выбрать правильные драйвера для вашей графической карты Nvidia. Будутвыбраны драйвера Bumblebee, если у вас гибридная карта (как Nvidia, так и Intel). Используйте «ddm -p nvidia», чтобы удалить драйвера из вашей системы.

# ddm -i open

Вы можете использовать эту команду, если хотите вернуться к открытым драйверам Nouveau. Это удалит все проприетарные драйвера из вашей системы.

# ddm -i broadcom

Эта команда установит правильный драйвер для вашего беспроводного адаптера Broadcom. Используйте «ddm -p Broadcom», чтобы удалить драйвера из вашей системы.

# ddm -i pae

Это позволит установить ядро ​​PAE на многопроцессорных системах под управлением 32-битной ОС. Используйте «ddm -p pae», чтобы удалить ядро ​​PAE из вашей системы. Обратите внимание, что вы не можете удалить ядро, загруженное в данный момент .
